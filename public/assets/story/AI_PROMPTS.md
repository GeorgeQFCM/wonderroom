# 故事卡片 AI 图片生成 Prompt

## 技术规格
- **尺寸**: 200x200 像素 (正方形)
- **格式**: PNG (透明背景)
- **风格**: Kawaii 卡通风格，适合4岁儿童

## 通用 Prompt 后缀
```
kawaii style, for children's educational game, simple design, vibrant colors,
round shapes, friendly expression, transparent background, 200x200 pixels,
centered composition, no text, child-friendly, 2D flat illustration
```

## 图片处理提示

1. **生成后处理**:
   - 调整尺寸到 200x200 像素
   - 确保背景透明
   - 优化文件大小 (< 50KB)

2. **推荐工具**:
   - Midjourney (--style cute)
   - DALL-E 3
   - Stable Diffusion + ControlNet

3. **后期处理工具**:
   - remove.bg (移除背景)
   - Photopea (免费在线 Photoshop)
   - ImageMagick (批量处理)
